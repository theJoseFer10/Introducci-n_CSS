Diseño responsive

Un ejemplo básico de Media Query en CSS usa @media para aplicar estilos condicionalmente, como cambiar el color de fondo a azul (background-color: blue;) cuando la pantalla es de 600px o menos (@media (max-width: 600px)), o un diseño diferente para impresión (@media print), permitiendo un diseño web responsivo y adaptable a distintos dispositivos 


    /* Estilos por defecto (para pantallas anchas) */
    body {
    background-color: lightblue;
    font-size: 16px;
    }

    /* Media Query para pantallas pequeñas (móviles) */
    @media (max-width: 600px) {
    body {
        background-color: steelblue; /* Cambia a azul oscuro */
        font-size: 14px; /* Fuente más pequeña */
    }
    h1 {
        color: white; /* Título blanco en móvil */
    }
    }
