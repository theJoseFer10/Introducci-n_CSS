Ejemplo de enlaza de un archivo css a html.

Podemos enlazar nuestros archivos css a nuestro archivo html de una manera muy sensilla, esto lo logramos mediante la etiqueta link. En la parte de abajo se muestra un ejemplo de uso de esta etiqueta. 

                    
                        <!DOCTYPE html>
                        <html>
                        <head>
                            <meta name="viewport" content="width=device-width, initial-scale=1.8">
                            <link rel="stylesheet" href="rutaCss">
                            <title>Mi Página</title>
                        </head>
                        <body>
                            <h1>¡Hola Mundo!</h1>
                        </body>
                        </html>
