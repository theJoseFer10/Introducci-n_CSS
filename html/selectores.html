Ejemplo de selectorees en css.

Con ayuda de los selectores podemos hacer referencia a elementos html y manipularlos en nuestro css, solo necesitamos en nuestro html una clase o un id. Y en nuestro css podemos referenciarlos con # para los id y . para las clases. 

                    
                        <!--HTML-->
                        <div id="contenedor">
                            Contenido de nuestro div.
                        </div>

                        <div clas="cajas">
                            caja 1
                        </div>

                        <div clas="cajas">
                            caja 2
                        </div>

                        <div clas="cajas">
                            caja 3
                        </div>

                        <div clas="cajas">
                            caja 4
                        </div>

                        <!--CSS-->

                        #contenedor{
                            Estislo para el div con el id "contenedor"
                        }

                        .cajas{
                            Estilo para los divs con la clase "cajas"
                            Se aplica a cada elemento html con esa clase.
                        }
