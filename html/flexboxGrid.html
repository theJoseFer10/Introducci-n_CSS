Ejemplo de flexbox.

Un ejemplo básico de Flexbox es centrar elementos en una barra de navegación o crear un grid de tarjetas, usando display: flex; en el contenedor para activar el layout flexible, y luego propiedades como flex-direction (para filas o columnas), justify-content (para el eje principal) y align-items (para el eje secundario) para organizar los elementos hijos (ítems), permitiendo diseños adaptables y alineación precisa sin float ni posicionamiento complejo 


<!--HTML de ejemplo.-->
<nav class="navbar">
    <a href="#"> Inicio <a>
    <a href="#"> Conocenos <a>
    <a href="#"> Mapa <a>
    <a href="#"> Comentarios <a>
    <a href="#"> Menu <a>
</nav>
<!--CSS de ejemplo.-->
.navbar {
    display: flex; /* Activa Flexbox */
    justify-content: space-around; /* Distribuye espacio entre elementos */
    align-items: center; /* Centra verticalmente */
    background-color: #333;
    padding: 10px;
}
.navbar a {
    color: white;
    text-decoration: none;
    padding: 5px 10px;
}

<!--HTMl de ejemplo.-->
<div class="card-container">
    <div class="card">
        <h3>Tarjeta 1</h3>
        <hp>Texto tarjeta 1</hp>
    </div>
        <div class="card">
        <h3>Tarjeta 2</h3>
        <hp>Texto tarjeta 2</hp>
    </div>
        <div class="card">
        <h3>Tarjeta 3</h3>
        <hp>Texto tarjeta 3</hp>
    </div>
        <div class="card">
        <h3>Tarjeta 4</h3>
        <hp>Texto tarjeta 4</hp>
    </div>
</div>

<!--CSS de ejemplo.-->
.card-container {
    display: flex;
    flex-wrap: wrap; /* Permite que las tarjetas pasen a la siguiente línea */
    gap: 20px; /* Espacio entre tarjetas */
    padding: 20px;
}
.card {
    flex: 1 1 300px; /* Crece, encoge, base de 300px */
    background-color: #f4f4f4;
    border: 1px solid #ddd;
    padding: 20px;
    box-shadow: 2px 2px 5px rgba(0,0,0,0.1);
}
